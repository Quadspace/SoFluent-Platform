# So Fluent Brand Guide

**Source:** [Brand Manual Folder](https://drive.google.com/drive/folders/108UeI-Mr564Q05WOUT8SMKyFvARBj69q)

## Brand Assets to Download

### Required Files:
1. **Manual de Marca** - Complete brand guidelines
2. **Logo** - Logo files (SVG, PNG formats)
3. **Padrões e Grafismos** - Patterns and graphics
4. **Fontes** - Brand font files
5. **Cartão de Visita** - Business card reference

## Implementation Notes

After downloading assets:
1. Place logo files in `client/src/assets/branding/logos/`
2. Place patterns in `client/src/assets/branding/patterns/`
3. Place fonts in `client/src/assets/branding/fonts/`
4. Update components to reference these assets
