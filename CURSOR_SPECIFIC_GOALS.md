## üéØ Code Generation Priorities

When generating code, prioritize:

1. **Architecture Compliance:** Follow the three golden rules
2. **Component Reusability:** Build components, not pages
3. **Theme Consistency:** Use `theme.ts` for all styling
4. **DRY Principle:** No code duplication
5. **Mobile-First:** Responsive by default
6. **Accessibility:** WCAG 2.1 AA compliance
7. **Performance:** Optimize for speed
8. **Internationalization:** Support EN/PT
